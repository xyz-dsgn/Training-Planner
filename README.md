# Training-Planner
Программа для планирования тренировок
# Название проекта Training-Planner
**Автор:** Елизавета Недолужко
**Вариант:** Training Planner
**Дата сдачи:** 04.05.2026
## Описание программы 
Графическое приложение на Python для планирования и отслеживания тренировок. Приложение позволяет добавлять, просматривать, фильтровать и удалять записи о тренировках, а также сохранять и загружать данные в формате JSON.
## Требования для запуска 
Что нужно установить на компьютер: 
- Python 3.10 или выше 
- Библиотеки: `pip install tkinter`
## Как запустить 
```bash 
git clone https://github.com/xyz-dsgn/Training-Planner
cd Training-Planner
python main.py
```
## Как пользоваться 
Добавление тренировки
- Введите дату в формате ГГГГ-ММ-ДД (например, 2024-12-25)
- Выберите тип тренировки из выпадающего списка
- Введите длительность в минутах (положительное число)
- Нажмите кнопку "Добавить тренировку"
## Пример работы 
1. Запуск программы — открывается окно с полями ввода и таблицей.
2. Добавление тренировки:
 - Дата: 2026-05-04
 - Тип: Бег
 - Длительность: 45
 - Нажать Добавить тренировку
3. Фильтр по типу:
 - Выбрать Бег → нажать Применить фильтр — показываются только тренировки с бегом.
4. Фильтр по дате:
 - Ввести 2026-05-04 → нажать Применить фильтр — остаётся только добавленная тренировка.
